Parish Administrator — How to add or update news posts

News is managed through the file data/news.json in this website folder. You do not need to edit any HTML. Simply open that file in a text editor (e.g. Notepad) and follow the format below.

Each post looks like this:

{
  "id": "5",
  "title": "Your News Headline Here",
  "date": "2025-07-15",
  "category": "Parish News",
  "excerpt": "A short one-sentence summary shown on the home page.",
  "body": "The full text of the article. Use \\n\\n for paragraph breaks.",
  "image": "https://example.com/your-image.jpg"
}

The "image" field is optional — simply leave it out if you have no image.
If included, paste any direct image link (see sources below) and it will appear at the top of the post.

Where to get an image link:

  • Your own website / Facebook: right-click an image → “Copy image address”
  • Google Drive: share the image → copy the link → change /file/d/FILE_ID/view to /uc?id=FILE_ID
  • Dropbox: share → copy link → change ?dl=0 to ?raw=1
  • Imgur: right-click the image on the page → “Copy image address”

Available categories: Parish News, Events, Sacraments, Upcoming, Notices.
Separate posts with commas. The most recent post (highest "date") will appear first.
To remove a post, delete its entire {…} block (and the comma before or after it).
Once saved and uploaded to GitHub, changes appear on the website immediately.
